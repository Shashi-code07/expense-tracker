# 💰 Expense Tracker

A modern and responsive **Expense Tracker Web Application** built using **HTML, CSS, JavaScript, and Chart.js**. It helps users manage their income and expenses, track monthly budgets, visualize spending with charts, and monitor financial activities through an interactive dashboard.

## ✨ Features

- 💰 Add Income & Expense Transactions
- ✏️ Edit Transactions
- 🗑️ Delete Transactions
- 🔍 Search Transactions
- 🎯 Filter by Type & Category
- 📊 Dashboard Summary
- 💳 Current Balance
- 📈 Total Income
- 📉 Total Expense
- 🏦 Total Savings
- 🎯 Monthly Budget Planner
- 📊 Budget Progress Bar
- ⚠️ Budget Warning
- 🔔 Toast Notifications
- 🥧 Doughnut Chart (Income vs Expense)
- 📊 Category-wise Expense Bar Chart
- 📈 Statistics Dashboard
- 🌙 Dark / ☀️ Light Mode
- 💾 Local Storage Support
- 📱 Fully Responsive Design
